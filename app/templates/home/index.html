This web app is mainly for practicing flask (a Python web Microframework), Python, and Jinja2.
Beside that, learning Linux and testing out different technologies and store the content on this site.

No Javascript is used, just Python, Flask, Jinja2, SQLite

testit.tech is powered by:

  • Nginx HTTP server and reverse proxy
  • Gunicorn Python WSGI HTTP server for UNIX
  • DigitalOcean Cloud computing, designed for developers (Linux)
  • doimainnameshop Domain at doimainnameshop
  • And a bunch of tutorials, and links for learning *

* Most important links / tutorials are found under "Links used" in upper right corner of the menu
