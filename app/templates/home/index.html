This web app is mainly for practicing flask (a Python Microframework), Python, Linux and testing out different technologies.

Sqlite is also used here for several tasks, amongst many for storing data for a Notebook

Notebook is an app where you can save / delete notes. A note has an id, note, link, topic, create-date and owner.

The "Technology" tab contains information and testing of different software and cmd's.

testit.tech is powered by:

  • Nginx HTTP server and reverse proxy
  • Gunicorn Python WSGI HTTP server for UNIX
  • DigitalOcean Cloud computing, designed for developers (Linux)
  • doimainnameshop Domain at doimainnameshop
  • And a bunch of tutorials, and links for learning *

* Most important links / tutorials are found under "Links used" in upper right corner of the menu
